# LeetCode 49. 字母异位词分组（Group Anagrams）

## 🎯 问题描述
给定字符串数组，将字母相同（即字母异位词）的字符串分组。

## 🌟 核心思路
字母异位词排序后一定相同，因此：

- 对每个字符串排序作为 key
- 使用 unordered_map<string, vector<string>> 分类
- 最后输出所有 value

时间复杂度：  
- 每个字符串排序 O(K log K)，K 为字符串长度  
- 总体 O(NK log K)

空间复杂度：O(NK)

---

## 🧩 关键原理
字母异位词 = 字符及数量完全一致。  
排序后即得到唯一表示（signature）。

例如：

| 字符串 | 排序后 | 分组 key |
|--------|--------|----------|
| eat    | aet    | aet      |
| tea    | aet    | aet      |
| ate    | aet    | aet      |

- `it->first` = key

## 📌 知识点总结
- 哈希表按“排序后的字符串”分组  
- unordered_map 更快（O(1) 均摊）  
- map/unordered_map 的迭代器是指向 pair 的指针  
- 使用 `it->second` 访问 value
